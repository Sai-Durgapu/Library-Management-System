# Library-Management-System


**Developer:** Durgapu Sai Krishna

## Overview

A simple Library Management System developed using C# and .NET.

## Features

* Add New Book
* Register Borrower
* Borrow Book
* Return Book
* View Books
* View Borrowers

## Testing

* MSTest Framework
* Unit Tests Implemented
* All Tests Passed

## Technologies Used

* C#
* .NET
* MSTest

## Run Project


dotnet build
dotnet test
